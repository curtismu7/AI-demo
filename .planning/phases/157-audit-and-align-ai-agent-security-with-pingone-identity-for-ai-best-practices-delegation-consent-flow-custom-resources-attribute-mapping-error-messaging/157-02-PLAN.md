- **Critical** — Security vulnerability or complete feature gap; agent delegation is broken or insecure
- **High** — Significant deviation from guide; delegation works but with reduced security
- **Medium** — Missing best practice; functionality works but doesn't follow PingOne patterns
- **Low** — Minor deviation or optimization opportunity; doesn't affect security posture
