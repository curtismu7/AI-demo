<objective>
Consolidate the two stray `:root { }` blocks into `index.css` so all global CSS custom properties live in one place.

Purpose: Having `:root` in 3 files (`index.css`, `dashboard-theme.css`, `FloatingPanel.css`) makes token discovery hard and risks ordering bugs. After this plan only `index.css` defines global tokens.

**What moves:**
1. `dashboard-theme.css` lines 5–22 (the `:root { --dash-* }` block, 13 vars) → append to `index.css` `:root` block
2. `FloatingPanel.css` lines 99–102 (`:root { --fp-edge: 5px; --fp-corner: 12px; }`, 2 vars) → append to `index.css` `:root` block

**What stays put:**
- `html[data-theme='dark'] { ... }` in `dashboard-theme.css` — NOT a `:root` block, do not touch
- All component-level rules in both files remain unchanged

Output: Both source files lose their `:root {}` blocks. `index.css` gains the vars.
</objective>
