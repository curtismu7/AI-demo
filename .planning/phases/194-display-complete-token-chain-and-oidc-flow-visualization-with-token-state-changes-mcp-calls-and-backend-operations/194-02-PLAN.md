<context>
## Token Color System Already Exists
TokenColorSystem.js provides:
- `getTokenColor(tokenType)` → returns color for user_token, agent_token, mcp_token
- `TokenColorDot` component (small colored circle with tooltip)
- `TokenColorLegend` component (shows all token types and meanings)
- `deriveTokenCategory(event)` → categorizes token event

## Token State Lifecycle
From TokenChainContext events and agentFlowDiagramService:
- `acquiring` — fetching token from OAuth server (spinner)
- `active` — token obtained, ready to use
- `exchanged` — token exchanged for another (e.g., user token → MCP token)
- `used` — token was used in an operation (e.g., sent to MCP server)
- `failed` — token acquisition/exchange failed (error state)

## Display Requirements
- Sub claim (user ID, truncated with friendly name if available)
- Act claim (agent ID if present, indicating delegation)
- May_act claim (permissions granted)
- Scopes (what the token can do)
- Expiry (time until token expires)
</context>

Create a React component that displays a token's current state inline. Compact inline display (~100x40px), expands on hover/click.

Component props:
```jsx
<TokenStateIndicator 
  token={{ 
    tokenType: 'user_token' | 'agent_token' | 'mcp_token',
    status: 'acquiring' | 'active' | 'exchanged' | 'used' | 'failed',
    sub: 'user-uuid',
    act: { sub: 'agent-uuid' } | null,
    may_act: [...],
    scopes: ['banking:read', ...],
    exp: timestamp
  }}
  resolvedIdentity={{ 
    currentUser: { sub, name, email }, 
    knownClients: { [clientId]: label } 
  }}
  compact={true}
/>
```

1. **Inline (compact=true)**: Colored dot + token type abbreviation (U/A/M) + state spinner/icon (acquiring=spinner, active=checkmark, exchanged=arrow, used=checkmark, failed=✕)
2. **Expanded (onClick/hover)**: Show full details:
   - Token type with color
   - Current state + acquired timestamp
   - Sub claim (with friendly name if resolvedIdentity.currentUser matched)
   - Act claim (with friendly name if resolvedIdentity.knownClients matched)
   - Scopes list
   - Expiry countdown (e.g., "expires in 5m 30s")

Use TokenColorSystem.getTokenColor() and TokenColorDot for consistency.
