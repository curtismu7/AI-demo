## Token Milestone States
From research: `{ id, timestamp, milestone_type, status: pending|active|done|error, details }`

Milestone types:
1. `oidc_login` — User initiated OAuth flow
2. `exchange_start` — Beginning token exchange (1 or 2)
3. `exchange_complete` — Token exchange finished, MCP token acquired
4. `mcp_tool_call` — Invoking MCP tool with MCP token
5. `backend_operation` — Banking API call (balance, transfer, etc.)
6. `flow_complete` — All operations finished
</context>
