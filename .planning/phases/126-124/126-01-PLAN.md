Implement Phase 126 so the token chain UI shows who the human user is and which app or agent is acting on that user's behalf. Replace opaque raw identifiers with friendly labels while preserving copyable raw IDs and safe fallbacks.

Output: TokenChainDisplay, education panels, and AgentFlowDiagramPanel surface friendly sub/act identity sourced from the current session or BFF profile lookup.
</objective>
