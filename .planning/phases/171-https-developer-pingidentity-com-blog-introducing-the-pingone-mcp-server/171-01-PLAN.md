## Project Context

BX Finance is an educational demo for developers showing secure AI agent integration using modern OAuth 2.0 patterns. The demo is live at banking-demo-puce.vercel.app and includes:
- Authorization Code + PKCE login
- CIBA (Backchannel Authentication Initiative) flows
- RFC 8693 token exchange (1-exchange and 2-exchange paths)
- MCP server with secure delegation via `act` claims
- HITL (Human-In-The-Loop) verification for sensitive operations
