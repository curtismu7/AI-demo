<objective>
Intercept `mcp_scope_denied` (HTTP 403) errors in the MCP tool call layer and automatically attempt a single RFC 8693 token exchange to get a scope-upgraded token, then retry the tool call. If the retry also fails, surface the error clearly. If the error is not scope-related, propagate immediately.

Purpose: Agents should silently recover from missing-scope errors rather than dead-ending the user. This is the agentic pattern for scope escalation — the LLM never sees the error.

Output: Modified `callMcpTool()` in `mcpToolRegistry.js` with scope upgrade + once-only retry.
</objective>

2. In `callMcpTool(toolName, params, agentToken, userId, tokenEvents = [])`, after the existing `const response = await fetch(...)` + `const data = await response.json()` block, add this scope upgrade block BEFORE the current `if (!response.ok)` throw:

   ```javascript
   // Scope upgrade: if MCP returned 403 mcp_scope_denied, attempt RFC 8693 token exchange once
   if (response.status === 403 && data?.error === 'mcp_scope_denied' && !_scopeUpgradeAttempted && agentToken) {
     _scopeUpgradeAttempted = true;
     const missingScopes = data.missingScopes || [];
     const availableScopes = data.availableScopes || [];
     const audience = process.env.MCP_SERVER_RESOURCE_URI;
     if (missingScopes.length > 0 && audience) {
       try {
         const newScopeList = [...new Set([...availableScopes, ...missingScopes])];
         const exchanged = await oauthService.performTokenExchange(agentToken, audience, newScopeList);
         const upgradedToken = exchanged?.access_token;
         if (upgradedToken) {
           if (tokenEvents) {
             tokenEvents.push({
               type: 'scope_upgrade',
               timestamp: new Date().toISOString(),
               tool: toolName,
               missingScopes,
               newScopes: newScopeList,
               actor: 'agent',
               note: 'RFC 8693 token exchange on behalf of scope upgrade',
             });
           }
           // Retry the tool call with the upgraded token (exactly once — _scopeUpgradeAttempted guard prevents loop)
           return callMcpTool(toolName, params, upgradedToken, userId, tokenEvents, true);
         }
       } catch (upgradeErr) {
         console.warn(`[MCP_TOOL] Scope upgrade failed for tool '${toolName}':`, upgradeErr.message);
         // Fall through to the error throw below
       }
     }
   }
   ```

   async function callMcpTool(toolName, params, agentToken, userId, tokenEvents = [], _scopeUpgradeAttempted = false) {
