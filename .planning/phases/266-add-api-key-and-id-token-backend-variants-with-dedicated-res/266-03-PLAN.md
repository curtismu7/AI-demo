The Token Chain narrative reflects this by including segments for "bearer validated" and "id_token decoded server-side at banking_resource_server." Plan 01 emits these synthesized events; this plan ensures they render correctly.

**R2 narrative for dual_token (from Plan 01 Task 2):** the gateway emits a 4-segment chain:
- `evt-inbound` — Inbound user bearer received
- `evt-idtoken-fetch` — id_token fetched from BFF session (server-to-server)
- `evt-bearer-validated` — banking_resource_server: bearer validated (authenticateToken middleware)
- `evt-idtoken-decoded` — banking_resource_server: id_token decoded server-side, claims returned (no raw JWT)

All four segments carry `credentialPath: 'dual_token'`. The TokenChainDisplay must render all of them with the teal visual identity.

```jsx
import { SPEC_GUIDE } from './specGuide';

function SpecRefPill({ specRef }) {
  const [expanded, setExpanded] = React.useState(false);
  // Multi-spec citations like "RFC 8693 + draft-ietf-oauth-identity-chaining" — split on " + " and render each
  const refs = specRef.split(' + ');
  return (
    <span className="tcd-specref-group">
      {refs.map((r) => {
        const entry = SPEC_GUIDE[r] || SPEC_GUIDE[specRef] || null;
        if (!entry) return <span key={r} className="tcd-specref-pill tcd-specref-unknown">{r}</span>;
        return (
          <a
            key={r}
            className="tcd-specref-pill"
            href={entry.url}
            target="_blank"
            rel="noopener noreferrer"
            onMouseEnter={() => setExpanded(true)}
            onMouseLeave={() => setExpanded(false)}
            onClick={(e) => { e.preventDefault(); setExpanded(!expanded); }}
            title={entry.title}
          >
            {r}
            <span className="tcd-specref-link-icon" aria-hidden>↗</span>
          </a>
        );
      })}
      {expanded && (
        <div className="tcd-specref-explainer">
          {refs.map((r) => {
            const entry = SPEC_GUIDE[r];
            return entry ? (
              <div key={r} className="tcd-specref-explainer-row">
                <strong>{entry.title}:</strong> {entry.summary}
                {' '}
                <a href={entry.url} target="_blank" rel="noopener noreferrer">read spec ↗</a>
              </div>
            ) : null;
          })}
        </div>
      )}
    </span>
  );
}
```
