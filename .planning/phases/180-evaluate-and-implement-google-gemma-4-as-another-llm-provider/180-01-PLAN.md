   - "withdraw 25 from checking" → banking/withdraw
   - "how does MCP work?" → education/mcp-protocol
   - "what are my accounts?" → banking/accounts

   - Send the intent through the same OpenAI-compatible chat completions endpoint format
   - Record: response time (ms), parsed result, whether it matches expected kind/action

   - `LM_STUDIO_BASE_URL` for local (default: http://localhost:1234/v1)

   ```
   ...
   Accuracy                        | 8/9        | 9/9       | 7/9
   ```

5. Script runs with: `node banking_api_server/scripts/compare-llm-intents.js`
