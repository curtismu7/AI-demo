GET /api/admin/vault/status → 200:
  { unlocked: boolean, entriesLoaded: number, vaultFilePresent: boolean, vaultPath: string }
  (vaultPath is basename only)

POST /api/admin/vault/unlock body { password: string } → 200:
  { ok: true, entriesLoaded: number }
Failure shapes:
  400 { error: 'bad_request', message: 'password (string) required' }
  401 { error: 'unauthorized', message: 'vault: open failed (bad password or tampered file)' }
  404 { error: 'vault_file_not_found' }
  429 { error: 'too_many_requests', message: 'Too many unlock attempts. Wait 5 minutes.' }
  500 { error: 'io_error', message: 'unlock failed' }
  503 { error: 'vault_disabled_serverless', message: '...' }

POST /api/admin/vault/rotate body { currentPassword: string, newPassword: string } → 200:
  { ok: true, message: 'Vault password rotated. Update VAULT_PASSWORD before next BFF restart.' }
Failure shapes:
  400 { error: 'bad_request' | 'weak_password' | 'same_password', message: '...' }
  401 { error: 'unauthorized', message: 'vault: open failed (bad password or tampered file)' }
  409 { error: 'rotate_in_progress' }
  423 { error: 'vault_locked', message: 'Unlock the vault before rotating.' }
  500 { error: 'io_error', message: 'rotate failed' }
  503 { error: 'vault_disabled_serverless', message: '...' }

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser DOM → BFF /api/admin/vault/* | httpOnly session cookie + JSON body; apiClient sets withCredentials:true |
| User keyboard → React component state | Passwords held in component-local useState; cleared on success; never stored in localStorage/sessionStorage |
| Render output → DOM | Password values only render through `<input type="password">`; never via {password} interpolation in JSX text |
