## Tests

### 1. Admin unlocks vault via web UI after BFF restart
expected: Operator starts BFF without `VAULT_PASSWORD` env, signs in as admin, navigates to `/admin/vault`, enters vault password, sees "vault unlocked" banner + status badge flips to ✅ unlocked. After unlock, `configStore.getEffective('helix_api_key')` (etc.) returns the vault-supplied value.
requirement: REQ-VAULT-ADMIN-04
how-to:
  - Stop BFF if running; unset `VAULT_PASSWORD` in shell
  - Start BFF: `./run-bank.sh` (or `cd banking_api_server && npm start`)
  - Sign in as admin user via PingOne
  - Navigate to `/admin/vault`
  - Confirm status badge shows "❌ locked" with entries_loaded = 0
  - Enter the vault password in the Unlock form, click Unlock
  - Confirm green ✅ banner appears
  - Confirm status badge flips to "✅ unlocked" with entries_loaded > 0
result: [pending]

### 2. Admin rotates password from web UI; restart with new password
expected: With vault unlocked, operator clicks Rotate, enters current + new password (twice for confirm), sees success banner. After rotate, restarting the BFF with the OLD `VAULT_PASSWORD` in env fails on startup; restarting with the NEW password succeeds. Proves rotate persisted to disk.
requirement: REQ-VAULT-ADMIN-05
how-to:
  - With vault unlocked from test 1, go to Rotate section
  - Enter current password, new password (twice), submit
  - Confirm green ✅ banner + ⚠️ "update VAULT_PASSWORD before next restart" note
  - Restart BFF with OLD `VAULT_PASSWORD` env — startup MUST fail with vault open error
  - Update env to NEW password; restart BFF — startup MUST succeed and entries load
result: [pending]

### 3. Opaque error UX on wrong password (no enumeration oracle)

requirement: REQ-VAULT-ADMIN-07

  - With vault locked, enter a deliberately wrong password
  - Confirm UI shows generic ❌ error banner (no "wrong password" vs "tampered file" disambiguation)
  - Open browser DevTools Network tab; inspect the POST /api/admin/vault/unlock response
  - Confirm response body is the same generic shape regardless of cause
