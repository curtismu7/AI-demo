# Plan 237-01 — Audit & Dead-Code Removal

## Objective

Remove dead files, eliminate inline styles from education panels, and identify any
components over 300 lines that have a clean split point. No behavior changes.

- [ ] Delete `banking_api_ui/src/components/BankingAgent.backup.css`
- [ ] Check for any other `*.backup.*` files in `src/`

### T2 — Inline style audit in education panels
Files to check for `style={{ ... }}` blocks:
- `education/RFC8693Panel.js`
- `education/TokenExchangePanel.js`
- `education/TokenChainPanel.js`
- `education/RFC9728Content.js`
