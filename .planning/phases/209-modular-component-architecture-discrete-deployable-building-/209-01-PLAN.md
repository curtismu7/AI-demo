Extract shared services into a new `banking-core/` local npm package. This makes the
dependency surface explicit and versioned, enabling future components to depend on shared
logic without cross-folder source imports.

must live in a dedicated package. Extraction uses re-export shims so no existing caller
anywhere in banking_api_server changes — backward compatibility is non-negotiable.

Output: `banking-core/` package with 3 extracted services; banking_api_server updated to
declare `banking-core` as a local dependency and to use re-export shims at the original
service paths.
