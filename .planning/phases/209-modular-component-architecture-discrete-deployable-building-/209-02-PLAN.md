against Okta/Keycloak set OIDC_* overrides; PingOne users set nothing new.

confirming the precedence chain (OIDC_* > PINGONE_* > derived); D-08 confirmed by adapter
not covering Management API endpoints.

    ```javascript
    /**
     * OIDCProviderAdapter.js
     *
     * Normalises PINGONE_* environment variables to a standard OIDC endpoint interface.
     * Engineers using a different OIDC provider (Okta, Keycloak, PingFederate) override
     * via OIDC_* env vars without changing code (per D-05, D-06, D-07).
     *
     * Precedence (highest to lowest):
     *   1. OIDC_* override env vars (deployment-time, non-PingOne IDPs)
     *   2. PINGONE_* env vars (default, existing deployments)
     *   3. Derived from configStore region + environmentId (PingOne default)
     *
     * D-08: Management API endpoints (PingOne Management, Authorize worker) are NOT
     * included here. Those remain PingOne-specific in their respective services.
     */
    'use strict';

    class OIDCProviderAdapter {
      /**
       * @param {object} configStore - The configStore singleton (from banking-core or shim)
       */
      constructor(configStore) {
        this._configStore = configStore;
      }

      /**
       * Build the base PingOne AS URL from region + environmentId.
       * @private
       */
      _pingOneAsBase() {
        const region = this._configStore.getEffective('pingone_region') || 'com';
        const envId = this._configStore.getEffective('pingone_environment_id') || '';
        if (!envId) return '';
        return `https://auth.pingone.${region}/${envId}/as`;
      }

      /**
       * OIDC issuer URL.
       * @returns {string}
       */
      get issuer() {
        const override = process.env.OIDC_ISSUER;
        if (override) return override;
        const base = this._pingOneAsBase();
        if (!base) throw new Error('OIDC issuer not configured: set OIDC_ISSUER or PINGONE_ENVIRONMENT_ID + PINGONE_REGION');
        return base;
      }

      get authorization_endpoint() {
        return process.env.OIDC_AUTHORIZATION_ENDPOINT
          || process.env.PINGONE_AUTHORIZATION_ENDPOINT
          || `${this.issuer}/authorize`;
      }

      get token_endpoint() {
        return process.env.OIDC_TOKEN_ENDPOINT
          || process.env.PINGONE_TOKEN_ENDPOINT
          || `${this.issuer}/token`;
      }

      get introspection_endpoint() {
        return process.env.OIDC_INTROSPECTION_ENDPOINT
          || process.env.PINGONE_INTROSPECTION_ENDPOINT
          || `${this.issuer}/introspect`;
      }

      get userinfo_endpoint() {
        return process.env.OIDC_USERINFO_ENDPOINT
          || process.env.PINGONE_USERINFO_ENDPOINT
          || `${this.issuer}/userinfo`;
      }

      get jwks_uri() {
        return process.env.OIDC_JWKS_URI
          || `${this.issuer}/jwks`;
      }
    }

    module.exports = { OIDCProviderAdapter };
    ```

    describe('OIDCProviderAdapter (BFF)', () => {
      const origEnv = { ...process.env };
      afterEach(() => {
        // Restore env
        Object.keys(process.env).forEach(k => {
          if (k.startsWith('OIDC_') || k.startsWith('PINGONE_AUTHORIZATION') ||
              k.startsWith('PINGONE_TOKEN') || k.startsWith('PINGONE_INTROSPECTION') ||
              k.startsWith('PINGONE_USERINFO')) {
            delete process.env[k];
          }
        });
        Object.assign(process.env, origEnv);
      });

      describe('issuer derivation', () => {
        it('returns OIDC_ISSUER when set (override path)', () => {
          process.env.OIDC_ISSUER = 'https://my-okta.okta.com/oauth2/default';
          const adapter = new OIDCProviderAdapter(makeConfigStore());
          expect(adapter.issuer).toBe('https://my-okta.okta.com/oauth2/default');
        });

        it('derives PingOne AS URL from region + envId when no OIDC_ISSUER', () => {
          delete process.env.OIDC_ISSUER;

          expect(adapter.issuer).toBe('https://auth.pingone.com/abc-123/as');
        });

        it('throws when neither OIDC_ISSUER nor envId are configured', () => {
          delete process.env.OIDC_ISSUER;
          const adapter = new OIDCProviderAdapter(makeConfigStore('com', ''));
          expect(() => adapter.issuer).toThrow('OIDC issuer not configured');
        });
      });

      describe('introspection_endpoint precedence', () => {
        it('OIDC_INTROSPECTION_ENDPOINT takes priority', () => {
          process.env.OIDC_INTROSPECTION_ENDPOINT = 'https://custom.idp.com/introspect';
          process.env.PINGONE_INTROSPECTION_ENDPOINT = 'https://pingone.com/introspect';
          const adapter = new OIDCProviderAdapter(makeConfigStore());
          expect(adapter.introspection_endpoint).toBe('https://custom.idp.com/introspect');
        });

        it('PINGONE_INTROSPECTION_ENDPOINT used when no OIDC_ override', () => {
          delete process.env.OIDC_INTROSPECTION_ENDPOINT;
          process.env.PINGONE_INTROSPECTION_ENDPOINT = 'https://auth.pingone.com/env/as/introspect';
          const adapter = new OIDCProviderAdapter(makeConfigStore());
          expect(adapter.introspection_endpoint).toBe('https://auth.pingone.com/env/as/introspect');
        });

        it('derives from issuer when no env vars set', () => {
          delete process.env.OIDC_INTROSPECTION_ENDPOINT;
          delete process.env.PINGONE_INTROSPECTION_ENDPOINT;
          const adapter = new OIDCProviderAdapter(makeConfigStore('com', 'abc-123'));
          expect(adapter.introspection_endpoint).toBe('https://auth.pingone.com/abc-123/as/introspect');

        fields.forEach(f => {
          expect(typeof adapter[f]).toBe('string');
          expect(adapter[f].length).toBeGreaterThan(0);
        });
      });
    });

    ```typescript
    /**
     * OIDCProviderAdapter.ts
     *
     * Normalises PINGONE_* environment variables to standard OIDC endpoint fields.
     * Supports OIDC_* override vars for non-PingOne IDPs (D-05, D-06, D-07).
     *
     * Precedence: OIDC_* > PINGONE_* > derived from PINGONE_BASE_URL
     *
     * D-08: Management API features remain PingOne-specific; this adapter covers
     * core OAuth/agent flows only.
     *
     * Note: This adapter is intentionally duplicated per-component (not shared via
     * banking-core) because banking_mcp_server is TypeScript and banking-core is
     * JavaScript. The interface contract is documented in banking-core/docs/.
     */

    export class OIDCProviderAdapter {
      /**
       * OIDC issuer URL.
       * Precedence: OIDC_ISSUER > PINGONE_BASE_URL + '/as' > ''
       */
      get issuer(): string {
        return process.env.OIDC_ISSUER
          || (process.env.PINGONE_BASE_URL
              ? process.env.PINGONE_BASE_URL.replace(/\/?$/, '/as')
              : '');
      }

      /**
       * Token introspection endpoint.
       * Precedence: OIDC_INTROSPECTION_ENDPOINT > PINGONE_INTROSPECTION_ENDPOINT > derived
       */
      get introspection_endpoint(): string {
        return process.env.OIDC_INTROSPECTION_ENDPOINT
          || process.env.PINGONE_INTROSPECTION_ENDPOINT
          || (this.issuer ? `${this.issuer}/introspect` : '');
      }

      /**
       * Token endpoint.
       * Precedence: OIDC_TOKEN_ENDPOINT > PINGONE_TOKEN_ENDPOINT > derived
       */
      get token_endpoint(): string {
        return process.env.OIDC_TOKEN_ENDPOINT
          || process.env.PINGONE_TOKEN_ENDPOINT
          || (this.issuer ? `${this.issuer}/token` : '');
      }

      /**
       * Authorization endpoint.
       * Precedence: OIDC_AUTHORIZATION_ENDPOINT > PINGONE_AUTHORIZATION_ENDPOINT > derived
       */
      get authorization_endpoint(): string {
        return process.env.OIDC_AUTHORIZATION_ENDPOINT
          || process.env.PINGONE_AUTHORIZATION_ENDPOINT
          || (this.issuer ? `${this.issuer}/authorize` : '');
      }

      /**
       * JWKS URI.
       */
      get jwks_uri(): string {
        return process.env.OIDC_JWKS_URI
          || (this.issuer ? `${this.issuer}/jwks` : '');
      }

      /**
       * UserInfo endpoint.
       */
      get userinfo_endpoint(): string {
        return process.env.OIDC_USERINFO_ENDPOINT
          || (this.issuer ? `${this.issuer}/userinfo` : '');
      }

      /**
       * Build a PingOneConfig-compatible object for TokenIntrospector and
       * BankingAuthenticationManager. Callers that currently pass a raw config
       * object built from environments.ts can switch to this method.
       *
       * @param clientId - PINGONE_CLIENT_ID (or OIDC_CLIENT_ID if set)
       * @param clientSecret - PINGONE_CLIENT_SECRET
       */
      toPingOneConfig(clientId: string, clientSecret: string): {
        baseUrl: string;
        clientId: string;
        clientSecret: string;
        tokenIntrospectionEndpoint: string;
        authorizationEndpoint: string;
        tokenEndpoint: string;
      } {
        return {
          baseUrl: process.env.PINGONE_BASE_URL || this.issuer,
          clientId,
          clientSecret,
          tokenIntrospectionEndpoint: this.introspection_endpoint,
          authorizationEndpoint: this.authorization_endpoint,
          tokenEndpoint: this.token_endpoint,
        };
      }
    }
    ```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Env var → OIDCProviderAdapter getter | Untrusted env vars could be empty or malformed URLs |
| OIDCProviderAdapter → external OIDC endpoints | Endpoint URLs used for token introspection and auth flows |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-209-02-01 | Spoofing | OIDCProviderAdapter returning empty introspection_endpoint | mitigate | BankingAuthenticationManager already validates token against endpoint; empty string causes HTTP error which is caught and returned as auth failure — not silent pass. Validate non-empty string at call site before invoking introspection. |
| T-209-02-02 | Spoofing | OIDC_* override vars pointing to attacker-controlled endpoints | accept | Env vars are deployment-time configuration set by the operator, not user input. Operators are trusted. Document in .env.example that OIDC_* vars must point to trusted OIDC servers. |
| T-209-02-03 | Elevation of Privilege | Malicious OIDCProviderAdapter implementation swapped in (D-12 adapter swap risk) | mitigate | OIDCProviderAdapter is instantiated once at server startup from source code, not from external config. No runtime swap path exists in this phase. Future adapter factory patterns must validate interface compliance. |
| T-209-02-04 | Information Disclosure | Bearer tokens logged if introspection_endpoint URL is logged alongside token calls | accept | Existing logging in TokenIntrospector.ts logs endpoint URL (not the token value). This is acceptable; token value is never logged per existing policy in REGRESSION_PLAN.md. |
</threat_model>
