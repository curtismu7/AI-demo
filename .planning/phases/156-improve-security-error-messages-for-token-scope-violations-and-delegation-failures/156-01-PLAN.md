# 8 Error Categories to Handle
1. Token Type Mismatch (user vs. agent vs. system token)
2. Scope Violation (missing required scopes)
3. Audience Mismatch (token audience != endpoint audience)
4. Missing Delegation Claim (no 'act' claim when required)
5. Token Expired (token.exp < now)
6. Rate Limit Exceeded (too many requests)
7. Insufficient Permissions (admin operation, user role)
8. Policy Violation (transaction limits, agent limits)

# Error Response Format (JSON Schema)
```json
{
  "error": "TOKEN_TYPE_MISMATCH",
  "message": "Token type error: This endpoint requires an 'agent' token...",
  "details": {
    "what_failed": "Token type validation",
    "why": "Security reason (3-4 sentences)",
    "teaching": "What this teaches about security",
    "tokens_involved": {
      "token_type": "user",
      "token_scopes": ["profile", "email"],
      "required_scopes": ["mcp:execute"],
      "token_aud": "https://api.mybank.com",
      "expected_aud": "https://mcp.mybank.com:8443"
    },
    "fix": "Specific action to resolve"
  },
  "documentation_link": "https://docs.mybank.com/errors/TOKEN_TYPE_MISMATCH"
}
```

# BFF Middleware Integration Points
- tokenValidator.js: Extract and validate token type
- scopeChecker.js: Validate required scopes
- delegationValidator.js: Check for 'act' claim
- rateLimit.js: Count requests
- adminGuard.js: Check for admin role

   ```javascript
   import ErrorSchemaService from '../services/errorSchemaService.js';
   import ErrorMessageBuilder from '../services/errorMessageBuilder.js';
   
   export default function tokenErrorMiddleware(requiredTokenType) {
     return (req, res, next) => {
       const token = req.user?.token;
       const actualTokenType = token?.token_type; // or custom claim
       
       if (!actualTokenType || actualTokenType !== requiredTokenType) {
         const details = ErrorMessageBuilder.buildTokenTypeMismatch(
           requiredTokenType,
           actualTokenType || 'unknown',
           token?.scopes || []
         );
         const errorResponse = ErrorSchemaService.buildErrorResponse(
           'TOKEN_TYPE_MISMATCH',
           { message: details.what_failed, ...details }
         );
         const statusCode = ErrorSchemaService.getStatusCode('TOKEN_TYPE_MISMATCH');
         return res.status(statusCode).json(errorResponse);
       }
       next();
     };
   }
   ```

2. **scopeErrorMiddleware.js** (detect scope violation):
   ```javascript
   import ErrorSchemaService from '../services/errorSchemaService.js';
   import ErrorMessageBuilder from '../services/errorMessageBuilder.js';
   
   export default function scopeErrorMiddleware(requiredScopes) {
     return (req, res, next) => {
       const tokenScopes = req.user?.scopes || [];
       const hasAllScopes = requiredScopes.every(scope => tokenScopes.includes(scope));
       
       if (!hasAllScopes) {
         const missingScopes = requiredScopes.filter(scope => !tokenScopes.includes(scope));
         const details = ErrorMessageBuilder.buildScopeViolation(
           requiredScopes,
           tokenScopes
         );
         const errorResponse = ErrorSchemaService.buildErrorResponse(
           'SCOPE_VIOLATION',
           { 
             message: details.what_failed,
             tokens_involved: { token_scopes: tokenScopes, required_scopes: requiredScopes },
             ...details
           }
         );
         const statusCode = ErrorSchemaService.getStatusCode('SCOPE_VIOLATION');
         return res.status(statusCode).json(errorResponse);
       }
       next();
     };
   }
   ```

3. **delegationErrorMiddleware.js** (detect missing 'act' claim):

       }
       next();
     };
   }
