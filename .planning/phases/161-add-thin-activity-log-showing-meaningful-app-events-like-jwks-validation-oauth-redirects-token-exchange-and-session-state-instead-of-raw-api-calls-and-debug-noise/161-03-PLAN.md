   - `mcp` → 🤖
   - `auth_lifecycle` → 🔐

4. **Severity styling:**
   - `info` → default/neutral
   - `warning` → amber/orange left-border or badge
   - `error` → red left-border or badge

5. **Timeline layout:** Vertical timeline with:
   - Left: timestamp (relative, e.g. "2m ago") with absolute time on hover
   - Center: category icon + severity indicator
   - Right: message text + expandable metadata
