<objective>
Add LM Studio as a local model provider fallback when Groq quota is exceeded (HTTP 429).

Purpose: Groq's free tier has rate limits. When exceeded, the demo should gracefully fall back to a local LM Studio instance (OpenAI-compatible API on localhost:1234) before trying Gemini or heuristic. This keeps the demo functional during development/presentations without cloud API dependencies.

Output: Updated BFF NL intent chain and LangChain agent factory with LM Studio support.
</objective>
