SSE via fetch (POST endpoint — cannot use EventSource which is GET-only):
```javascript
// Use fetch + ReadableStream for POST SSE
const res = await fetch('/api/dev/run-servers', {
  method: 'POST',
  credentials: 'include',
});
if (res.status === 409) {
  setStatus('already_running');
  return;
}
if (res.status === 403) {
  setStatus('forbidden');
  return;
}
const reader = res.body.getReader();
const decoder = new TextDecoder();
let buffer = '';
while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  buffer += decoder.decode(value, { stream: true });
  const events = buffer.split('\n\n');
  buffer = events.pop(); // incomplete chunk
  for (const event of events) {
    const dataLine = event.replace(/^data: /, '').trim();
    if (!dataLine) continue;
    try {
      const parsed = JSON.parse(dataLine);
      handleEvent(parsed);
    } catch (_) {}
  }
}
```

Status card heuristic (simple — no brittle regex):
```javascript
function detectService(line) {
  if (line.includes(':3001') || line.toLowerCase().includes('banking api')) return 'API Server';
  if (line.includes(':4000') || line.toLowerCase().includes('banking ui')) return 'UI Server';
  if (line.includes(':8080') || line.toLowerCase().includes('mcp')) return 'MCP Server';
  if (line.includes(':8888') || line.toLowerCase().includes('langchain')) return 'LangChain Agent';
  return null;
}
```

Create `RunServersModal.js` — a React functional component. Key behaviors per D-03:

**State:**
- `lines` — array of `{ text, type: 'stdout'|'stderr' }` objects (raw log)
- `cards` — object keyed by service name, value `{ name, detected: bool }` (status cards)
- `status` — `'streaming'|'done'|'error'|'already_running'|'forbidden'`
- `exitCode` — number or null
- `autoDismissCountdown` — number (3..0) only when `status === 'done'`

**On mount:** fire the fetch stream immediately (no button click inside modal — the modal opening IS the trigger). Use the fetch+ReadableStream pattern from `<interfaces>`.

**handleEvent(parsed):**
- `{ line, type: 'stdout'|'stderr' }` → append to `lines`; check `detectService(line)` and add/update `cards`
- `{ type: 'done', exitCode: 0 }` → set `status='done'`; start 3-second countdown; auto-call `onClose` at 0
- `{ type: 'error', exitCode }` → set `status='error'`; set `exitCode`

**Auto-dismiss countdown:** Use `useEffect` + `setInterval` when `status === 'done'`. Decrement `autoDismissCountdown` each second; call `onClose()` when it reaches 0.
