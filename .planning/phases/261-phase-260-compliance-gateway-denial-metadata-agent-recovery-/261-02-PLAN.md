## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-261-04 | Elevation of Privilege | agentOrchestrator.ts -32403 branch | mitigate | Recovery only throws an error — no automatic token retry or re-authentication from agent side. The BFF controls any token refresh flow. Agent cannot bypass auth by catching its own error. |
