<objective>
Add a BFF scope-upgrade endpoint that performs an RFC 8693 token exchange explicitly including
`banking:write` and caches the result in the session as `mcpWriteToken`. Modify `POST /api/mcp/tool`
to use the cached write token for write tools, bypassing the normal exchange on replay.

Purpose: When the client presents the scope-upgrade consent modal and the user approves, the UI
calls `POST /api/mcp/scope-upgrade`. The BFF exchanges the user token specifically for a
write-capable MCP token and stores it. The subsequent `runAction()` replay then finds the cached
token and succeeds without re-triggering a 403.

Output: `/api/mcp/scope-upgrade` route in server.js + mcpWriteToken fast-path in POST /api/mcp/tool.

    ```js
    // POST /api/mcp/scope-upgrade — RFC 8693 exchange for banking:write scope
    // Called by the UI after the user approves the scope-upgrade consent modal.
    // Stores the write-scoped MCP token in req.session.mcpWriteToken (session-cached, one exchange per session).
    app.post('/api/mcp/scope-upgrade', express.json(), requireSession, async (req, res) => {
        try {
            // Use 'create_transfer' as the tool for scope resolution — any write tool resolves banking:write.
            const { token, tokenEvents } = await resolveMcpAccessTokenWithEvents(req, 'create_transfer');
            if (!token) {
                return res.status(403).json({
                    error: 'scope_upgrade_failed',
                    message: 'Token exchange did not return a write-scoped token. Check agent_mcp_allowed_scopes includes banking:write.',
                    tokenEvents,
                });
            }
            // Cache write-scoped MCP token in session — reused by POST /api/mcp/tool for write tools.
            req.session.mcpWriteToken = token;
            await new Promise((resolve, reject) =>
                req.session.save(err => err ? reject(err) : resolve())
            );
            return res.json({ ok: true, tokenEvents });
        } catch (err) {
            console.error('[/api/mcp/scope-upgrade] Exchange failed:', err.message);
            return res.status(500).json({
                error: 'scope_upgrade_failed',
                message: err.message,
            });
        }
    });
    ```
