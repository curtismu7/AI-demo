<objective>
Fix two embedded-mode bugs that cause DevToolsDashboard tabs to appear broken:

1. **UnifiedTokenFlowInspector (UTFI) close bug**: When embedded in DevToolsDashboard with `showToggle={false}`, the `×` button calls `setVisible(false)` permanently — no toggle exists to reopen the panel. The fix: hide the `×` close button when `showToggle={false}` (the tab is the dismiss mechanism).

2. **McpTrafficPage 401 error**: On unauthenticated pages (LandingPage) or before session is established, polling `/api/mcp/traffic` returns 401 which displays as "HTTP 401" in red. The fix: check for 401 specifically and show a friendly "Sign in to view MCP traffic" state instead.

3. **DevToolsDashboard overflow audit**: Verify each tab panel's `overflow` and `height` is correct so content isn't clipped.

Purpose: Users opening the DevTools Dashboard should see usable, non-broken UI in all three tabs.
Output: Fixed UTFI embedded mode, fixed McpTrafficPage 401 state, confirmed DevToolsDashboard layout.
</objective>
