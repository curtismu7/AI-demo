<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Sidebar → Router | Menu items trigger navigation; malformed routes could break app |
| Sidebar → onClick handlers | Direct calls to onLogout, onThemeToggle, etc.; handlers must be safe |
| User role display | Menu items hidden/shown based on user.role; must validate role before rendering admin items |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-155-01 | Elevation of Privilege | Admin menu item visibility | mitigate | Check user.role === 'admin' before rendering admin items; validate role from auth context (not user input) |
| T-155-02 | Information Disclosure | Token Inspector access | mitigate | Token Inspector button only appears if user authenticated; don't leak token content in sidebar |
| T-155-03 | Denial of Service | onClick handlers | mitigate | Add loading states to async actions (role switch); prevent multiple clicks with disabled flag |
| T-155-04 | Tampering | Logout handler | mitigate | Ensure onLogout clears session properly; sidebar passes through, not responsible for cleanup |

</threat_model>
