# Sketch Manifest

## Design Direction
Clean chat-first floating agent. All chips and power controls hidden behind a single "Actions" pill button in the header. Header stripped from 7 controls to 3 (Actions trigger, Expand, Collapse). Left rail removed — chat area gets full width. Banking dark-blue color palette (`#152a52` / `#1a3a6e` / `#4169e1`).

## Reference Points
Existing BankingAgent.js/CSS (exact colors, fonts, and interaction patterns preserved)
