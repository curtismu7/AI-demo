Placement — inline on dashboard, FAB elsewhere

The Middle/Float/Always-float toggle is removed. On /dashboard the command bar above IS the agent, full width. On /marketing, /admin, /configure, etc., the same agent collapses to the FAB pictured at lower right — same composer, same suggestions, same token-chain drawer, in a 360-px popover.

Why this works — Option 1: Command Bar

  • One job per surface. The agent is the only first-class card. Everything else collapses into a single drawer.
  • Toolbar dies. Theme / layout / reset / token timer fold into a single 28-px eyebrow with icon buttons + ⌘ shortcuts. From 8 visible controls → 4 icons.
  • Real composer. The "where do I type?" question is answered the moment you land — a focused input is the most prominent element.
  • Suggestions, not CTAs. Monospaced chips with a verb color cue read as completions, not as competing buttons.
  • Token Chain becomes inspectable, not omnipresent. It's still one keystroke (⌘T) away — just not eating 40 % of the viewport.
  • One placement rule. Inline on dashboard, FAB everywhere else. No user toggle, no "always float" checkbox.
