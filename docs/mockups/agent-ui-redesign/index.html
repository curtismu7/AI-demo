Agent UI — three redesign options

Each option commits to one placement on the dashboard and shows the same agent compressed to a FAB on other pages. The current Middle / Float / Always-float toggle is retired in every option — no user choice, one consistent answer per design.

Option 1 · inline composer

Command bar

Single composer is the hero. All chrome collapses into one 28-px eyebrow with ⌘ shortcuts. Token chain, rules, tools become collapsed drawers.

  • Dashboard: inline command bar (full width)
  • Other pages: 360-px composer popover behind a small dark FAB
  • Best for: making the existing layout grown-up
 Option 2 · inline split

Split canvas (dark ops)

Chat left, live token-chain narration right. Theme/layout/reset collapse to a single 3-tab segmented control. Editorial serif + acid accent.

  • Dashboard: 60/40 split — chat on left, narration on right
  • Other pages: chat-only FAB; "open inspector" jumps to dashboard
  • Best for: showcasing the token-flow story to technical audiences
 Option 3 · floating pill

Floating pill

Whole agent is a 56-px Raycast-style pill at the bottom of the dashboard. Dashboard reclaims the page. Pill expands in place into a panel with tabs.

  • Dashboard: persistent 720-px pill, center-bottom
  • Other pages: same agent compressed to a 56-px circular FAB
  • Best for: when banking-first feel is the priority
